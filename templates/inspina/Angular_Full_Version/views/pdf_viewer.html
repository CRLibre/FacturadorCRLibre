PDF.js

PDF.js is a Portable Document Format (PDF) viewer that is built with HTML5. PDF.js is community-driven and supported by Mozilla Labs. The goal is to create a general-purpose, web standards-based platform for parsing and rendering PDFs. Full documentation for angular directive: https://github.com/sayanee/angularjs-pdf
