jsTree / ngJsTree is jquery plugin, that provides interactive trees. It is absolutely free, open source and distributed under the MIT license. jsTree is easily extendable, themable and configurable, it supports HTML & JSON data sources and AJAX loading. Get more info on: http://www.jstree.com/ and check documentation angular directive https://github.com/ezraroi/ngJsTree
