About the project

Inspinia Laravel 5 Seed Project is a special starter project for all users who want's to start developing with Laravel and don't want to wait for full version. It is a core Inspinia project with layout, example views and styles. If you plan to work on Laravel then this project should save you some time on initial integration work.

This documentation provide simple instruction how you can run the project. Please note that in future I will provide full version for Laravel 5 and this documentation will be upgraded with more detailed description.

Project description

First run

Project is build like a standard Laravel project with npm and artisan and gulp.
Once you unzip the archive please install all dependencies:

npm install

php composer.phar install

To run the project you just need to run artisan server:

php artisan serve

The whole project files are located under app/ and resource folder.
The structure is easy. Main view components are located in resource/views/ folder.
You can find main layout file: app.blade.php as well as footer.blade.php and navigation files.

In the resource/assets/ folder I place the main Inspinia and vendor files (css and js files).

Next important folder is app/Http/Controllers. In this folder you can find all controller I use for example to add two views.
routes/ folder has a web.php file where you can find definition for routes. Other folders are core Laravel project folders

Please note that this is a starter project so it is no a full version. It should help you start with Inspinia and Laravel and save you some time on initial integration.
