About the project

Inspinia Angular CLI 2 Seed Project is a special starter project for all users who want's to start developing with Angular 2 and don't want to wait for full version. It is a core Inspinia project with layout, example views and styles. If you plan to work on Angular 2 then this project should save you some time on initial integration.

This documentation provide simple instruction how you can run the project. Please note that in future I will provide full version for Angular 2 and this documentation will be upgraded with more detailed description.

Project description

First run

Project is build with Angular CLI, npm and webpack module.
Once you unzip the archive please install all dependencies:

npm install

In the package.json file you can find all available commands but most important are:

ng serve - this will start development version. Once you run it you can enter to the app in the browser on http://localhost:4200
npm run build - this will build version and copy it to the dist folder.

The whole project files are located under src/ folder.
The structure is easy. Main Angular modules are located in app/ folder.
You can find main layout files at /app/components/common/layouts/ as well as routes: app.routes.ts and of course module and component declaration: app.module.ts and app.component.ts.

In the assets/ folder I place the main Inspinia style.css.

Please note that project is based on Angular CLI so you can use different commands to scaffold:

Scaffold Usage
Component ng g component my-new-component
Directive ng g directive my-new-directive
Pipe ng g pipe my-new-pipe
Service ng g service my-new-service
Class ng g class my-new-class
Interface ng g interface my-new-interface
Enum ng g enum my-new-enum
Module ng g module my-module

For more information pelase visit https://github.com/angular/angular-cli
