About the project

Inspinia ReactJS Seed Project is a special starter project for all users who want's to start developing with ReactJS and don't want to wait for full version. It is a core Inspinia project with layout, example views and styles. If you plan to work on ReactJS then this project should save you some time on initial integration work.

This documentation provide simple instruction how you can run the project. Please note that in future I will provide full version for ReactJS and this documentation will be upgraded with more detailed description.

Project description

First run

Project is build with npm and webpack module.
Once you unzip the archive please install all dependencies:

npm install

In the package.json file you can two main commands:

npm start - this will start development version on your localhost.
npm run build build - this will build version and copy it to the dist folder.


The whole project files are located under app/ folder.
The structure is easy. Main React render component is located in app/ index.js file.
All shared component like footer, navigation, layout you can find in app/components/common folder
Routes configuration you can find in app/config/routes.js file.
View files are located in app/views/ folder.

In the public/ folder I place the main Inspinia and vendor files (css and js).

Other folders are core ReactJS project folders.

Please note that this is a starter project so it is no a full version. It should help you start with Inspinia and ReactJS and save you some time on initial integration.
